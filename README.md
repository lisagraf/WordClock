# Flashing the ESP
1. Download the Arduino IDE
2. Download the board manager (TODO)
3. Select "Tools" in top bar
    a. Select Board NodeMCU 1.0 (ESP-12E Module)
    b. Set "Erase Flash" to "All Flash Contents"
    c. Select correct port (USB)
